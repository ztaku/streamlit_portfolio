
### myenvという名前の新しい仮想環境を作成
```bash
python -m venv myenv
```
### 仮想環境をアクティベート
```bash
myenv\Scripts\activate
```
### 仮想環境を無効化
```bash
deactivate
```

### アプリの実行
```bash
streamlit run app.py
```

### 必要なパッケージをインストール
```
pip install -r requirements.txt
```

Python 3.11で仮想環境を作り直す方法


C:\Users\user\AppData\Local\Programs\Python\Python311



# 仮想環境を削除（必要に応じて）
```
rmdir /s /q myenv
```
# Python 3.11 で仮想環境を作成
```
py -3.11 -m venv myenv
```
# 仮想環境をアクティベート
```
myenv\Scripts\activate
```
### 必要なパッケージをインストール
```
pip install -r requirements.txt
```
